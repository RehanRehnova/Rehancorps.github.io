Install

Step 1–2 are the same for every browser. Step 3 depends on Chrome-family vs Firefox.

  1. 1

    Download

    One zip. No account required.

    Download rehnova-passfill.zip
  2. 2

    Unzip

    Extract so you have a folder containing manifest.json. Keep that folder (don’t delete it after install).

    Example: Documents\rehnova-passfill

  3. 3

    Load in your browser

    Chrome · Edge · Brave · Arc

    1. Open extensions:
      chrome://extensions
      Edge: edge://extensions · Brave: brave://extensions
    2. Turn Developer mode ON
    3. Load unpacked → select the unzipped folder
    4. Pin Passfill

    Firefox

    1. Open:
      about:debugging#/runtime/this-firefox
    2. Click Load Temporary Add-on…
    3. Select manifest.json inside the unzipped folder
    4. Open the puzzle menu → pin Passfill

    Firefox temporary add-ons clear when you fully quit Firefox — reload the same folder after a restart, or later publish to AMO for a permanent install (free). Needs Firefox 115+.

First run

  • Signup form → focus password → ⚡ gen
  • When asked → Save · next visit → Fill
